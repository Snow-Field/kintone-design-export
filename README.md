# アプリ設計書出力ツール（Chrome拡張機能版）

kintoneアプリの設計情報をExcel形式で出力するChrome拡張機能です。<br>
[Gusuku Deploit]のような設計書スタイルを参考に、アプリの構成を一枚のExcelファイルにまとめます。

---

## 概要

kintoneアプリの管理・引き継ぎ・ドキュメント整備において、設計情報を手作業でまとめるのは手間がかかります。<br>
この拡張機能はkintoneのアプリ画面を開いた状態でボタンを押すだけで、アプリの設計情報をExcelファイルとして自動出力することが可能です。

---

## インストール方法

以下の手順でインストールできます。

1. `kintone-design-export.zip` ファイルを解凍する
2. Chromeで `chrome://extensions` を開く
3. 右上の **「デベロッパーモード」** をONにする
4. **「パッケージ化されていない拡張機能を読み込む」** をクリックし、解凍した `kintone-design-export` フォルダを選択する

---

## 使い方

※拡張機能追加前に開いたタブには動作しないので、ご注意ください。

1. kintoneのアプリ画面（`https://*.cybozu.com/k/{アプリID}/`）を開く
2. ツールバーの拡張機能アイコンをクリック
3. **「設計書出力」** ボタンを押す
4. `{ドメイン}-{アプリID}-{日付}.xlsx` が自動ダウンロードされる

---

## 出力シート一覧

| シート名 | 内容 |
|---|---|
| 一般情報 | アプリ名・アプリID・説明・ドメイン |
| フィールド一覧 | フィールドコード・名称・タイプ・必須/重複・デフォルト値・選択肢など |
| ルックアップ | コピー元アプリ・コピー元フィールド・フィールドマッピング |
| 関連レコード一覧 | 参照先アプリ・条件・表示フィールド |
| 一覧 | 一覧名・タイプ・表示フィールド・絞込条件・ソート条件 |
| アプリのアクセス権 | ユーザー/グループ/組織ごとの閲覧・追加・編集・削除・管理権限 |
| レコードのアクセス権 | 条件ごとのレコード閲覧・編集・削除権限 |
| フィールドのアクセス権 | フィールドごとの閲覧・編集権限 |
| プロセス管理 | ステータス・作業者・アクション・遷移条件 |
| 自動計算フィールド | 計算式を持つフィールドの一覧 |

---

## 動作要件

- Google Chrome
- kintoneのアプリ画面（`https://*.cybozu.com/k/*`）で実行すること
- アプリの設定を参照できる権限（アプリ管理者権限）
